# 🌟 Astreal Client (1.21.11 Fabric)

The definitive all-in-one Minecraft client bridging the gap between Lunar, Feather, and Badlion. 

## 🚀 Features
- **Performance:** Native Fabric 1.21.11 with Sodium & Lithium integration.
- **PvP Suite:** Crystal Optimizer, NoHurtCam, and custom Hitbox overlays.
- **Visuals:** Clean HUD, smooth animations, and Gen Z aesthetics.
- **Flexibility:** Full support for any third-party Fabric mods.
